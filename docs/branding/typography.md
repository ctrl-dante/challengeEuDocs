# Typography

Typography supports the **voice of ChallengeEU** — **clear, confident, and professional**.  
This section introduces the selected typefaces and their usage rules to ensure **consistency and legibility** across all brand materials.

---

## Primary Typeface: Alegreya

**Alegreya** is a humanist serif with a calligraphic feeling, conveying a **dynamic and varied rhythm**. Perfect for long-form content and editorial layouts.

> **Typeface available for download:** [Alegreya – Google Fonts](https://fonts.google.com/specimen/Alegreya)

### Weights & Styles

| Weight | Example |
|--------|--------|
| **Regular** | Alegreya Regular |
| *Regular Italic* | *Alegreya Regular Italic* |
| **Medium** | Alegreya Medium |
| *Medium Italic* | *Alegreya Medium Italic* |
| **Bold** | **Alegreya Bold** |
| *Bold Italic* | ***Alegreya Bold Italic*** |
| **ExtraBold** | Alegreya ExtraBold |
| *ExtraBold Italic* | *Alegreya ExtraBold Italic* |




## Implementation (Web)

```html
<link href="https://fonts.googleapis.com/css2?family=Alegreya:wght@400;500;700;800&family=Alegreya+Sans:wght@300;400;500;600;700;800&display=swap" rel="stylesheet">
```


## Paragraphs – Source Sans 3

**Source Sans 3** (formerly Source Sans Pro) is Adobe’s first open-source typeface family.It is a **sans-serif typeface** optimized for **user interfaces**, offering excellent legibility at small sizes and across digital screens.

> **Typeface available for download:** [Source Sans 3 – Google Fonts](https://fonts.google.com/specimen/Source+Sans+3)

### Recommended Use
- **Body text**
- **UI labels & buttons**
- **Forms & input fields**
- **Captions & metadata**

### Weights & Styles

| Weight | Example |
|--------|--------|
| **Light** | Source Sans 3 Light |
| *Light Italic* | *Source Sans 3 Light Italic* |
| **Regular** | Source Sans 3 Regular |
| *Regular Italic* | *Source Sans 3 Regular Italic* |
| **Medium** | Source Sans 3 Medium |
| *Medium Italic* | *Source Sans 3 Medium Italic* |
| **SemiBold** | Source Sans 3 SemiBold |
| *SemiBold Italic* | *Source Sans 3 SemiBold Italic* |
| **Bold** | **Source Sans 3 Bold** |
| *Bold Italic* | ***Source Sans 3 Bold Italic*** |

## Implementation (Web)

```html
<link href="https://fonts.googleapis.com/css2?family=Source+Sans+3:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```
